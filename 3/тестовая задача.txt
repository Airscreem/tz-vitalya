3) Создать контроллер для приложения на angularjs, который будет представлять из себя карусель слайдов. Подключать ангуляр не нужно, ровно как и выводить карусель в html, важен только один файл с контроллером, оформленный по всем правилам. За слайды принимаем некие абстрактные объекты которые добавляются в контроллер через метод addSlide и удаляются через метод removeSlide. Карусель должна иметь изменяемые параметры: 
а) интервал авто прокрутки
б) активация/деактивация авто прокрутки
